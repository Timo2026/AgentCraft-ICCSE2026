# AgentCraft — Agent开发教学 Copilot

> ICCSE 2026 参赛作品 | Education 赛道
> 标语: "教大学生从零搭建自己的AI助手"

## 解决的问题

学生学完AI课程后，仍然不会把大模型变成能干活的应用。
AgentCraft 是一个能教人一步步搭建Agent系统的Copilot。

## 核心工作流

```
学生需求 → 意图分析 → 骨架规划 → 组件搭建 → 门禁配置 → 回路测试 → 交付
```

## 六步教学法

1. **Understand** — 理解学生想造什么Agent
2. **Plan** — 拆解为skill/tool/gate组件
3. **Scaffold** — 生成Agent骨架代码
4. **Wire** — 连接意图路由+工具调用
5. **Gate** — 配置权限和门禁规则
6. **Test** — 运行测试用例验证
